4)What are CSS combinators?

Ans:-CSS combinators are explaining the relationship between two selectors. CSS selectors are the patterns used to select the elements for style purpose. A CSS selector can be a simple selector or a complex selector consisting of more than one selector connected using combinators.. CSS combinator selectors are as follows.
1.Descendant Selector (space)
2.Child Selector (>)
3.Adjacent Sibling Selector (+)
4.General Sibling Selector (~)

5)What are pseudo selectors in CSS?

Ans:- CSS pseudo-classes are used to add styles to selectors, but only when those selectors meet certain conditions. A pseudo class is expressed by adding a colon (:) after a selector in CSS, followed by a pseudo-class such as "hover", "focus", or "active".
